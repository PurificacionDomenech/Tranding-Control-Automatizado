# Trading Control Automatizado

## Project Overview
This is an automated trading registration application built with Flask and Python.

**Current State**: Basic Flask web application initialized and ready for development.

## Project Description
"Aplicacion para registro de trading, automatizada" - An automated trading registration application that will help users track and manage their trading operations.

## Technology Stack
- **Backend**: Python 3.11 + Flask 3.0
- **Frontend**: HTML/CSS with Jinja2 templates
- **Production Server**: Gunicorn

## Project Structure
```
├── app.py                 # Main Flask application
├── templates/             # HTML templates
│   └── index.html        # Landing page
├── static/               # Static assets
│   └── style.css         # Styles
├── requirements.txt      # Python dependencies
└── .gitignore           # Git ignore rules
```

## Features (Planned)
- Automatic trade registration
- Performance tracking
- Strategy analysis
- Transaction history

## Development
The application runs on port 5000 using the Flask development server.
Access the application through the Replit webview.
